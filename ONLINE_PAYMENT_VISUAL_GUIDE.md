## Security Features Visible

1. **Modal Dialog**
   - Prevents accidental submission
   - Shows all details before processing
   - Clear "Proceed" vs "Cancel" options

2. **Read-only Fields**
   - All student details shown but not editable
   - Prevents manual tampering
   - Pre-filled from server

3. **Secure Gateway**
   - Redirects to trusted CCAvenue HTTPS endpoint
   - Uses merchant credentials
   - Encrypted communication

4. **Audit Trail**
   - Transaction ID visible
   - Bank reference number stored
   - Timestamp recorded
   - Source marked as "Online Payment"
